# tr

#### 1. 更换字符

```bash
cat sample.txt | tr '0-9' '9876543210'
```

将`0`替换成`9`，`1`替换成`8`，`2`替换成`7`，依次类推。

#### 2. 压缩连续出现的相同字符

```bash
cat sample.txt | tr -s ' '
```

将连续多个空格压缩为一个

#### 3. 删除指定的字符

```bash
cat sample.txt | tr -d '70'
```

删除`7`和`0`两个字符串


#### 4. 使用补集

```bash
cat sample.txt | tr -c -d '135'
```

删除除了`1`、`3`和`5`以外的字符
