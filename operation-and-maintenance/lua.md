# Lua

安装Lua解释器
---

```
sudo apt-get install apt-file
sudo apt-file update
sudo apt-get install libreadline-dev


curl -R -O http://www.lua.org/ftp/lua-5.3.2.tar.gz
tar zxf lua-5.3.2.tar.gz
cd lua-5.3.2
make linux test install
```
