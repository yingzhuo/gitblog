# Groovy

#### 1. 下载
在groovy官方页面[下载](http://www.groovy-lang.org/download.html)二进制分发包。

#### 2. 安装
将`groovy-binary-2.4.6.zip`解压到`/usr/local/groovy-2.4.6`

#### 3. 配置

```
export GROOVY_HOME=/usr/local/groovy-2.4.5
export PATH=$PATH:$GROOVY_HOME/bin
```

#### 4. 测试

```bash
groovy -v
```

如果能看到gdk版本则说明安装成功了。
